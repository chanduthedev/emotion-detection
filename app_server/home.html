Emotion Detection API

This emotion detection API is REST-ful and supports CORS, so it can also be used in web browsers.

Routes

GET / (text/html)

Returns an HTML document with API routes and usage.

GET /predict (application/json)

Returns a emotion of the given text.

Query parameters:

  • text: Input text to predict the emotion.
  • model: (Optional) Model names are LR, SVM, NB and RF, default is LR.

Response:

  • text: Given text.
  • emotion: Predicted Emotion.
  • model: Used model for prediction.
